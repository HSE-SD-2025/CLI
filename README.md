# CLI Интерпретатор

## Описание проекта

Этот проект представляет собой простой интерпретатор командной строки (CLI), который поддерживает базовые команды, такие как `cat`, `echo`, `wc`, `pwd`, `exit`. Дополнительно поддерживает работу с переменными окружения и обеспечивают возможность вызова внешних программ, а также пайплайны для передачи данных между командами.

## Архитектура

Основные компоненты системы:

- **CLI Engine**: Основной компонент, отвечающий за обработку ввода пользователя, разбор строк и выполнение команд.
- **Команды**: Интерфейс команд с реализациями для каждой поддерживаемой команды (`EchoCommand`, `CatCommand`, `WcCommand`, `ExternalCommand`).
- **Переменные окружения**: Средство хранения и управления окружением с поддержкой подстановок.

Диаграмма структуры:

```plaintext
CLI Engine
├── Commands
│   ├── Command Interface
│   │   ├── EchoCommand
│   │   ├── CatCommand
│   │   ├── WcCommand
│   │   └── ExternalCommand
```
Подробности архитектуры описаны в [ARCHITECTURE.md](ARCHITECTURE.md).

## Начало работы

### Установка

1. **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/HSE-SD-2025/CLI.git
    cd CLI
    ```
2. **Сборка проекта:**

   Используйте Gradle для сборки проекта:
    ```bash
     ./gradlew build
    ```
### Запуск
Для запуска интерпретатора выполните:
    ```
    ./gradlew run
    ```
### Поддерживаемые команды

- `cat [FILE]`: Выводит содержимое файла на экран.
- `echo [ARGS]`: Выводит аргументы на экран.
- `wc [FILE]`: Выводит количество строк, слов и байт в файле.
- `pwd`: Показывает текущую директорию.
- `exit`: Выходит из интерпретатора.

Работа с одинарными и двойными кавычками, переменными окружения и оператором `|` для создания пайплайнов также поддерживается.
