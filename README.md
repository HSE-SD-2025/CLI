## Описание
Простой интерпретатор командной строки (CLI), поддерживающий базовые команды (cat, echo, pwd, wc, grep, exit), работу с переменными окружения, пайплайны через символ | и вызов внешних программ.

## Архитектура
CliEngine — основной управляющий компонент, реализующий цикл ожидания и обработки пользовательских команд.
Переменные окружения — хранятся в словаре внутри CliEngine и подставляются в команды на этапе разбора строки.
Команды реализуют интерфейс CommandInterface:
Все встроенные команды находятся в пакете commands:
EchoCommand, CatCommand, WcCommand, PwdCommand, GrepCommand, ExitCommand, а также ExternalCommand (для внешних вызовов).
Пайплайны — последовательная обработка команд, вывод предыдущей команды передаётся на ввод следующей.
Структурная диаграмма:

```plaintext
CliEngine
├── commands
│   ├── CommandInterface
│   │   ├── EchoCommand
│   │   ├── CatCommand
│   │   ├── WcCommand
│   │   ├── PwdCommand
│   │   ├── GrepCommand
│   │   ├── ExitCommand
│   │   └── ExternalCommand
Подробное архитектурное описание — см. ARCHITECTURE.md.
```
Как запустить
Клонируйте репозиторий

```bash
git clone https://github.com/HSE-SD-2025/CLI.git
cd CLI
```
Сборка

```bash
./gradlew build
```
Запуск

```bash
./gradlew run
```
(Или используйте другой подходящий командный запуск для вашего языка/окружения)

Использование
1. echo [аргументы] — выводит аргументы на экран
2. cat [файл] — выводит содержимое файла
3. wc [файл] — считает строки, слова, байты в файле
4. pwd — показывает текущую директорию
5. grep [шаблон] — ищет строки по шаблону из stdin или файла
6. exit — завершает работу CLI
7. VAR=значение — установка переменной окружения (доступно подстановкой: $VAR)
Кавычки — поддерживаются одинарные и двойные кавычки, корректно обрабатываются (например, аргументы с пробелами)
Пайплайны (|) — связывайте команды, чтобы направить результат одной как вход другой:
```bash
cat data.txt | grep hello | wc
```
Внешние команды — если команда не является встроенной, CLI попытается вызвать её как внешнюю (например: ls, date)
